1. Informações Coletadas

1.1 Dados fornecidos pelo usuário

  • Nome;
  • E-mail;
  • CPF;
  • Outras informações inseridas voluntariamente no aplicativo.

1.2 Dados de localização

Coletamos dados de localização do dispositivo, mediante autorização do usuário, para oferecer funcionalidades específicas do aplicativo.

A coleta pode ocorrer em:

  • Em tempo real, enquanto o aplicativo estiver em uso;
  • Em segundo plano, quando necessário para o funcionamento adequado de determinadas funcionalidades.

1.3 Dados de uso

  • Interações dentro do aplicativo;
  • Funcionalidades acessadas;
  • Tempo de uso e comportamento de navegação.

1.4 Dados do dispositivo

  • Modelo do dispositivo;
  • Sistema operacional;
  • Identificadores únicos;
  • Endereço IP.

2. Uso das informações

As informações coletadas são utilizadas para:

  • Garantir o funcionamento correto do aplicativo;
  • Personalizar a experiência do usuário;
  • Enviar notificações e atualizações;
  • Melhorar funcionalidades e desempenho;
  • Monitorar segurança e prevenir fraudes;
  • Cumprir obrigações legais.

3. Notificações

O aplicativo pode enviar notificações (push notifications) com:

  • Informações importantes sobre o uso do sistema;
  • Atualizações e avisos relevantes;
  • Comunicados relacionados ao serviço.

O usuário pode desativar as notificações a qualquer momento nas configurações do dispositivo.

4. Compartilhamento de dados

Não vendemos os dados pessoais dos usuários.:

Os dados podem ser compartilhados apenas quando necessário:

  • Com serviços de terceiros que auxiliam na operação do aplicativo (como serviços de banco de dados e hospedagem);
  • Para cumprimento de obrigações legais;
  • Para proteção dos direitos, segurança e integridade do aplicativo e de seus usuários.

O usuário pode desativar as notificações a qualquer momento nas configurações do dispositivo.

5. Armazenamento e segurança dos dados

Os dados são armazenados em servidores seguros e protegidos por medidas técnicas e administrativas adequadas. Adotamos práticas para evitar acesso não autorizado, vazamentos ou uso indevido das informações. Os dados serão mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política.

6. Base legal (LGPD)

O tratamento de dados é realizado com base nas seguintes hipóteses legais:

  • Consentimento do usuário;
  • Execução de contrato;
  • Cumprimento de obrigação legal;
  • Legítimo interesse, quando aplicável.

7. Direitos do usuário

Nos termos da Lei Geral de Proteção de Dados (LGPD), o usuário pode:

  • Solicitar acesso aos seus dados;
  • Corrigir dados incompletos ou desatualizados;
  • Solicitar a exclusão de seus dados;
  • Revogar o consentimento;
  • Solicitar a portabilidade dos dados.

Para exercer esses direitos, entre em contato conosco.

8. Cookies e tecnologias similares

Podemos utilizar cookies e tecnologias semelhantes para melhorar a experiência do usuário e analisar o uso do aplicativo.

9. Alterações nesta política

Esta Política de Privacidade pode ser atualizada a qualquer momento. Recomendamos a revisão periódica deste documento.
